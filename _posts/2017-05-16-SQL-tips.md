# This is the basic queries to collect data from SQL.
## Data collecting
```SQL
SELECT * FROM DATA.TABLE;
```
## Check NULL:
```SQL
ISNULL
NULLIF
```
## Sort
TOP N / TOP N Percent: select N rows or N percentage
OFFSET: limit
FETCH: get
```SQL
SELECT TOP ... FROM DATA.TABLE
ORDER BY ... ASC/DESC
OFFSET ... FETCH ...;
```
